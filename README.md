# Devil's Path

Devil's Path is a mobile platformer game built with Cordova.

## Features
- Challenging platformer gameplay
- AdMob integration for monetization
- Built for Android

## Tech Stack
- Apache Cordova
- HTML5/Canvas
- AdMob Plus

## Setup
1. Install dependencies: `npm install`
2. Add Android platform: `cordova platform add android`
3. Build: `cordova build android`
